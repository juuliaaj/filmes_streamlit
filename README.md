# Dashboard Netflix
### Instalação das Dependências

```bash
pip install streamlit pandas plotly matplotlib
```

### Como Executar
1. Execute o comando:

```bash
python -m streamlit run app.py
```

2. Acesse no navegador: http://localhost:8501
